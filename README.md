## HetznerWebserver
  
 - Nginx Webserver, with LetsEncrypt hetzner DNS certs
 - Debian/Ubuntu
 - Uses docker
 - Menu driven

  
  
  
#### Install

	Install Docker

    git clone https://github.com/logicethos/HetznerWebserver
    cd HetznerWebserver

Set up `sites-available`  `sites-enabled` and `www`

    ./menu
